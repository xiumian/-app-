# 宠伴记隐私政策
宠伴记用于宠物档案、打卡、提醒、护理记录、成长胶囊、暖窝动态、云同步、云备份、图片上传、账号导出和注销，由实际部署并对外提供服务的一方负责运营并公示联系方式。未配置远端 API 时数据仅在本浏览器；登录后账号、宠物、记录、图片、动态、评论、反馈与投诉、云状态、备份和请求 ID 会发往服务端，用于注册登录、同步、备份、图片上传、图片删除、账号导出和注销。同步体和备份不得包含 token、密码、cookie 或临时弹层状态。头像和成长胶囊可能保存到服务端；删除成长胶囊、宠物档案或注销账号时会清理可识别的远端媒体文件。远端账号可通过 GET /account/export 导出资料、应用状态和云备份，不得包含密码哈希、token 哈希、cookie 或私钥；也可通过 DELETE /account 注销，需再次输入密码，成功后删除当前用户、会话、云状态、备份和可识别媒体文件，并使旧 token 失效。服务端密码使用带盐哈希保存，token 只保存哈希值。诊断包不得包含用户内容、token、密码、cookie、Authorization 或密钥。你可联系运营者导出、更正、删除数据或撤回同意。健康提醒、护理记录和体重趋势仅用于日常管理，不构成兽医诊断或治疗建议。
