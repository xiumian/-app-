# 宠伴记用户协议
## 服务范围
宠伴记提供宠物档案、打卡、健康提醒、护理记录、成长胶囊、暖窝动态、云同步、云备份、媒体上传、账号导出和注销，由实际部署并对外提供服务的一方负责运营并公示联系方式。用户应填写合法资料，不得上传侵权、违法、骚扰、欺诈、恶意代码、隐私或无关内容，不得干扰他人数据。健康提醒、护理记录和体重趋势仅用于日常照护，不构成兽医诊断或治疗建议。未配置 API 时数据仅在本浏览器；配置远端账号后会与服务端交互。用户可导出账号数据、提交反馈与投诉或注销账号；注销会删除当前用户、会话、云端状态、备份和可识别媒体文件，并使旧 token 失效。应用会保存同意版本、时间和来源；未同意前不创建本地用户、远端账号、远端会话或演示数据。协议重要变更时应用提示确认。
